
- JSON server: is a lightweight and easy to use node.js tool that simulate a restful api using a JSON file as the data source.

Steps to create JSON Server
---------------------------

1) Create a folder for server
2) Create package.JSON file "npm init -y"
3) Install JSON Server Library "npm i json-server"
4) Create .ignore file ".gitignore"
5) to create JSON file to store JSON data
6) create index.js file (it is the only file that runs in the backend)
    - because it is the file that mention mainkey
    - entry point

require keyword is used to import json-server library

Middleware - it controls its function which has ability to control request function
